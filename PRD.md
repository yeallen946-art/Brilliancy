# PRD — Brilliancy

iOS 国际象棋"猜大师着法"训练应用 · 产品需求文档 v1.0
2026-06-07 · Owner: Jerry

---

## 1. 一句话定位

每天 5 分钟,跟大师下一盘名局:猜出大师的下一步,AI 用人话告诉你「为什么你的着法不如他的」。

## 2. 背景与机会

- 国际象棋手游市场 2025 年约 3.7 亿美元/年,头部(Chess.com)垄断,但细分训练品类有独立 app 的生存空间。
- "Guess the move" 玩法已被验证(ChessTempo、Forward Chess 免费提供,App Store 已有同类付费 app),但**全市场没有一家解释"为什么你的着法更差"**——引擎只给 ±0.3 这种冰冷数字。
- 差异化 = AI 自然语言讲解(锚定引擎输出)+ 进步追踪 + 每日挑战的仪式感。

## 3. 目标用户

**核心画像:改善型玩家(Improver)**

- 等级分 800–2000(USCF/Chess.com),知道"打谱学名局"有用但觉得枯燥
- 看 GothamChess / Hikaru,做战术题,但棋力停滞
- 痛点:打谱没反馈、不知道自己学到没有、坚持不下来

**种子用户:** Jerry 的儿子(USCF 1700),即代表性用户,承担可用性评审(见 §7)。

**非目标用户:** 纯新手(不懂规则)、2200+(需要的深度本产品不提供)。

## 4. 产品结构(混合型)

```
免费漏斗:每日挑战(拉新、留存、传播)
    ↓
付费核心:训练库(题库 + AI 讲解 + 进步追踪)→ 订阅
```

### 4.1 每日挑战(免费)

- 每天全球用户猜**同一盘名局**(精选 20–30 个关键着法点)
- 完成后出总分 + 可分享成绩卡(类 Wordle 表情格,不剧透具体着法)
- 连胜 streak,断签提示
- 历史每日挑战归档 → 付费解锁

### 4.2 训练库(付费核心)

- 按棋手分包:Tal 攻杀包、Morphy 包、Fischer 包、Carlsen 包……
- 按主题分包:王翼进攻、残局技术、防守反击……
- 每局含 AI 讲解(见 §6)
- MVP 内容量:**精讲 50 局**(每局全程可猜 + 全程讲解),宁缺毋滥;后续每月增补

### 4.3 进步追踪(付费)

- **棋感评分(Guess Rating)**:类 Elo,随每次猜测更新,有历史曲线
- 弱点分析:按局面类型(战术/局面/残局/开局)分维度统计命中率
- 评分曲线刻意调得前期略宽容,让用户两周内看到上涨(续订动机)

## 5. 核心玩法定义

**猜着法循环:**

1. 展示局面(用户执大师一方),给出对局背景(选手、年份、赛事)
2. 用户在棋盘上走出自己的猜测
3. 评分(三档):
   - **命中**(= 大师实着):满分
   - **同等好棋**(引擎评估差 ≤ 0.3 且不劣于实着):高分,并提示"你的着法同样成立,大师选择了另一条路"
   - **偏差**:按引擎评估差扣分;严重失误(≥ 1.5)标红
4. 展示大师实着 + AI 讲解(为什么这步好 / 为什么你的那步不如它)
5. 自动推进到下一个猜测点(对手着法快速播放)

**关键体验原则:**

- 不是每步都猜——开局照搬阶段快进,只在"有内容可讲"的着法点停下(每局 15–30 个点,内容流水线标注)
- 单局时长控制在 5–10 分钟
- 猜错不羞辱:讲解的语气是教练,不是判官

## 6. AI 讲解(核心差异化)

**定义:** 对每个猜测点,预生成自然语言讲解,覆盖:实着的意图、主要威胁/计划、以及针对最常见的 2–3 种错误猜测的"为什么不行"。

**硬性规则(产品层面):**

1. 讲解**必须锚定引擎输出**(Stockfish 评估值 + 主变),LLM 只负责把引擎变着翻译成人话,禁止自由发挥棋理
2. **预生成、不实时**:内容流水线离线批量生成,控制成本、保证质量(技术细节见 TECH_SPEC)
3. 语言水平对齐 1200–1800 读者:讲计划和威胁,不堆变着

**质检流程(双层):**

- 正确性 → 引擎自动校验(讲解中的着法必须在引擎 top moves / 主变内,评估描述与数值一致)
- 有用性 → 代表性用户评审(1700 棋手通读:读得懂吗?讲到点子上了吗?像人话吗?)抽查比例 MVP 期 100%,稳定后 ≥ 20%

## 7. 变现

**模式:Freemium 订阅 + 买断**

| 层级 | 价格 | 包含 |
|---|---|---|
| 免费 | $0 | 每日挑战、3 局体验精讲局、引擎对错判定 |
| 订阅 | $4.99/月 或 $29.99/年 | 全部题库、AI 讲解、棋感评分与弱点分析、每日挑战归档 |
| 买断 | $49.99 | 同订阅(棋类用户吃 Chessable 式买断逻辑) |

**原则:** 付费的是**讲解和进步追踪**,不是"更多棋局"(棋谱是公版的,量不值钱)。

- 苹果小企业计划:抽成 15%
- 试用:订阅 7 天免费试用
- 转化触点:每日挑战结束后展示"想知道为什么第 14 步你错了?"→ 引导订阅

## 8. 留存与增长

**留存:**

- 每日挑战 + streak(核心日活引擎)
- 每日推送一条有故事性的标题:"今天:Fischer 13 岁的世纪之局(1956)"
- 棋感评分涨跌反馈

**增长(获客内建在产品里):**

- 分享卡:每日挑战成绩卡,一键分享(iMessage / 社交),不剧透
- ASO 关键词:chess training, guess the move, chess puzzles, master games
- 冷启动渠道:r/chess、Chess.com 论坛、棋类 Discord、给中小chess YouTuber 送 lifetime code

## 9. 成功指标

| 阶段 | 指标 | 目标 |
|---|---|---|
| 验证期(TestFlight) | 种子用户(儿子+棋友)主动连续使用 | 7 天不催自玩 |
| 上线 3 个月 | 下载 | 3,000+ |
| 上线 3 个月 | D7 留存 | ≥ 15% |
| 上线 3 个月 | 免费→付费转化 | ≥ 2% |
| 上线 6 个月 | MRR | $500+(≈ 回本数倍) |

## 10. 风险与对策

| 风险 | 等级 | 对策 |
|---|---|---|
| Chess.com 抄功能 | 接受 | 只压天花板不影响副业目标;护城河=讲解质量+每日社区性 |
| AI 讲解一本正经胡说 | **高,必须管** | 引擎锚定 + 双层质检(§6);这是产品生死线 |
| 做完没人知道 | 高 | 分享卡内建传播;发布周主动社区发帖(§8) |
| 内容产能不足 | 中 | MVP 只做 50 精讲局;流水线自动化程度决定后续速度 |
| 留存崩(玩两天腻) | 中 | 每日挑战+streak;TestFlight 期重点观察 |

## 11. 非目标(明确不做)

- 人机对弈 / 在线对弈(Chess.com 的地盘,不碰)
- 开局背谱训练(Chessable 的地盘)
- Android(验证成功后再说)
- 社区/评论功能(运营成本高,V1 不做)
- 中文界面(先英文市场,棋类付费用户在欧美)
